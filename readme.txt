# Rendu du projet d'Advanced Supervised Learning
Auteurs : Antoine Gourru, Erwan Giry-Fouquet

Sujet choisi : Random Forest

Le rendu est composé d'un rapport, sous le format PDF, ainsi que de 4 fichiers :
2 fichiers sources :
- decisionTree.r : implémentation de l'arbre de décision
- RandomForest.r : implémentation du random forest ( nécessite decisionTree.r )

2 fichiers tests :
- decisionTree.test.r : tests unitaires validant les composants logiciels de l'arbre de décision
- RandomForest.test.r : tests et benchmark du RandomForest développé

Afin d'exploiter l'implémentation fournie, charger en mémoire le code du fichier RandomForest.r. (via source par exemple)

De plus, sont fournis différents jeux de données utilisés pour tester le logiciel.
